GymPay - Maqueta de interfaces (HTML + CSS)
------------------------------------------
Contenido:
- index.html -> Inicio de sesión (simulado). Redirige al panel (admin.html).
- register.html -> Registro (simulado). Redirige al panel (admin.html).
- admin.html -> Panel de administración (home).
- clients.html -> Registro y lista de clientes.
- memberships.html -> Gestión de membresías.
- payments.html -> Procesamiento de pagos.
- reports.html -> Generación de reportes (exportar simulado).
- history.html -> Consulta de historial del cliente.
- notifications.html -> Gestión de notificaciones.
- access_control.html -> Control de acceso (simulado).
- invoices.html -> Gestión de facturas.
- plans.html -> Creación de planes.
- attendance.html -> Registro de asistencia.
Instrucciones: descomprime y abre index.html en tu navegador. Los formularios son simulados y muestran alertas o redirigen según corresponda.
